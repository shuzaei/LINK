# LINK

落ちてくるミノを操作して, 環をつないで消すパズルゲームです.  

このゲームは筑波大学ゲーム制作サークル [Amusement Creators](https://www.amusement-creators.info/) にて, 
20210502 - 20210504 に開催された 48 時間ゲームジャムでの成果物です.  

同梱されている `readme.md` をお読みなってからご利用ください.
